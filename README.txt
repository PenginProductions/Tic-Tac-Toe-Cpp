This C++ project was created using Microsoft Visual Studio Community Edition. This program is a 2 player version of  Tic-tac-toe.
 4 files are contained:

	Assignment3_tictactoe.pdf => instructions given for the project during the c++ course.

	tictactoe_C++_assignment_sample.exe => sample .exe file provided by instructor to help in project creation.

	tictactoe.cpp => contains code for game functions as well as main game loop.

	Tic-Tac-Toe.exe => used to run program.
